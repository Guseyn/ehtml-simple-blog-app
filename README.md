# Simple [EHTML](https://github.com/Guseyn/EHTML) Blog App

```bash
git clone git@github.com:Guseyn/ehtml-simple-blog-app.git
cd ehtml-simple-blog-app
npm i
npm run app
```

Then go to http://localhost:8000/
